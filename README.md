# Machine Learning Projects Repository

This repository contains a collection of **machine learning and computer vision notebooks** implemented using Python. The projects focus on core ML algorithms, dimensionality reduction, classification techniques, and basic OpenCV-based applications. Each notebook is self-contained and demonstrates both theory and practical implementation.
